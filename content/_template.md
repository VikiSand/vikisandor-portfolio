Fields that are recognised:
  title       — can also be set with # heading at the top
  category    — design / research / teaching / publications / presentations
  tags        — comma-separated list
  year        — e.g. 2024
  video       — path or URL to video file (used as thumbnail)
  image       — path or URL to image (fallback if no video)
  link        — URL to the project detail page
  id          — slug used in URLs (auto-generated from filename if omitted)

Anything starting with _ is ignored by the sync script.
